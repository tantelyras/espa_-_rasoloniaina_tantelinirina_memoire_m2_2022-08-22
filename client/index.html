About

Facial Recognition system

The ability to map facial features mathematically and store data as face prints to uniquely identify and verify a face based on 128 values is Facial Recognition. It’s a biometric based software application technology that functions based on nodal points of a human face. The values associated with these nodal points help to identify or verify each unique face.
